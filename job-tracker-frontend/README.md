# 🚀 AI Job Tracker

An AI-powered job tracking web application that helps users analyze how well their resume matches job descriptions.

## 🔥 Features

* 🔐 JWT Authentication
* 💼 Job Application Tracking
* 🤖 AI-based Job Matching (Gemini API)
* 📊 Match Score, Missing Skills, Suggestions
* 🎯 Job-based AI analysis (not standalone)

## 🛠 Tech Stack

* Frontend: React
* Backend: Node.js, Express
* Database: MongoDB
* AI: Google Gemini API

## 📌 How It Works

1. User logs in
2. Adds job applications
3. Clicks "Analyze Job"
4. AI compares resume with job description
5. Displays:

   * Match Score
   * Missing Skills
   * Suggestions
